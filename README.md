# BC-GOV-CMS-Did-You-Find-Log-Parser
This is intended to process the BC Government's CMS "Did You Find?" logs, which can be found at: https://cmslite.gov.bc.ca/didyoufind_reports/gov/

Download and run "dyf_processor.exe" file. By default it expects to be in a folder that contains a subfolder called "Logs", which contains the "Did You Find?" logs.
